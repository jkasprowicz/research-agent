# Writing Style

## Goal

Produce mature, publication-level scientific writing.

## Preferred Characteristics

- Natural scientific tone
- Clear reasoning
- Strong cohesion
- Objective language
- Technical precision
- Humanized academic writing

## Avoid

- Generic AI phrasing
- Excessive adjectives
- Overly dramatic claims
- Repetitive sentence structures
- Artificial transitions
- Empty scientific jargon

## Discussion Style

Discussions should:
1. Interpret findings
2. Compare with literature
3. Explain mechanisms
4. Discuss limitations
5. Present realistic implications

Avoid merely repeating results.
